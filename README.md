# pneumonia-detection-cnn
Deep learning-based pneumonia detection from chest X-ray images using CNN.

## Overview
This project detects pneumonia from chest X-ray images using a deep learning model.

## Features
- Image preprocessing
- Model training
- Pneumonia prediction
- Performance evaluation

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Pandas
- OpenCV
- Matplotlib

## Dataset
Chest X-ray dataset used for training and testing.

## Results
The model classifies X-ray images as:
- Normal
- Pneumonia
